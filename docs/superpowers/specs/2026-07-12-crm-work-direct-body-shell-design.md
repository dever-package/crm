# CRM 工作台直接采用 Bot Body 外壳设计

## 目标

将 CRM 前台工作台的通用 `app-sidebar + app-topbar` 外壳完整替换为 `bot/body` 同款结构与视觉。保留 CRM 已有工作台、线索池、客户列表的真实路由、权限和业务功能，不再复用此前的 `work-skin` 样式覆盖方案。

## 已确认范围

- 桌面端使用 `bot/body` 的左侧导航、顶部当前页面标题和白色内容框。
- 左侧保留三个 CRM 入口：工作台、线索池、客户列表。
- 支持桌面侧栏折叠。
- 移动端使用底部导航，与 `bot/body` 的响应式方式一致。
- 删除当前通用顶栏、全局搜索和通用侧栏。
- 字体、字号、颜色、间距、背景、边框和圆角直接以 `bot/body` 为基准。
- 线索池、客户列表和工作台继续使用现有数据请求、弹窗和业务动作。
- 不修改后台管理端，不修改服务端业务模型和接口。

## 方案选择

### 采用：自定义壳组件 + Dever 路由出口

CRM 主页面改为新的工作台壳布局。壳组件负责品牌、导航、折叠状态、当前页面标题和响应式样式，页面 JSON 继续放置 `app-outlet`，因此 URL、浏览器前进后退、页面权限和现有子页面均保持有效。

### 不采用：继续覆盖通用壳样式

通用 `app-sidebar` 和 `app-topbar` 的 DOM 结构、尺寸与 `bot/body` 不同。继续增加 CSS 覆盖只能得到近似外观，并且容易被通用组件更新破坏。

### 不采用：单个 React 组件内部切页

直接复制 `WorkHomeShell` 的本地状态切页虽然最像示例，但会绕过 CRM 的真实路由、权限和浏览器历史，不符合现有系统结构。

## 组件边界

### `work-shell.tsx`

仅负责工作台外壳：

- 读取站点名称和 Logo。
- 根据当前路径标记活动菜单并显示页面标题。
- 使用 Dever 导航能力跳转真实页面。
- 保存桌面侧栏折叠状态。
- 提供直接改编自 `bot/body` 的工作台壳样式。

壳组件不请求 CRM 业务数据，也不管理线索或客户状态。

### `work/main.json`

负责组合外壳区域与 `app-outlet`。不再挂载 `app-sidebar`、`app-topbar`、`app-assistant` 或 `show-crm-work-skin`。

### 业务页面

工作台、线索池和客户列表保留现有组件和 API。页面根节点去除上一轮为样式覆盖添加的语义类，内容区样式只在新的工作台壳作用域内按 `bot/body` 重新定义。

## 导航与标题

| 路径 | 菜单 | 标题 |
| --- | --- | --- |
| `/crm/stats` | 工作台 | 工作台 |
| `/crm/lead` | 线索池 | 线索池 |
| `/crm/work` | 客户列表 | 客户列表 |

详情页或子路径沿用所属菜单的活动状态；无法匹配时显示站点名称，不阻断页面渲染。

## 视觉基准

直接沿用 `bot/body` 的主要参数：

- 页面背景 `#f4f6f5`，内容表面 `#ffffff`。
- 正文 `#171a19`，弱文字 `#6b7370`。
- 普通边框 `#e4e8e6`，强调边框 `#d2d9d6`。
- 桌面侧栏宽度 `240px`，主区域外边距 `11px 11px 11px 0`。
- 内容框圆角 `6px`，顶部标题栏高度 `38px`。
- 系统字体栈，基础字号 `12.8px`，字间距为 `0`。

不引入新颜色主题，不保留上一轮的卡片、阴影、表头和按钮覆盖规则。

## 状态与异常

- 路由跳转失败时不清空当前页面，保留现有 Dever 路由错误处理。
- Logo 或站点名称缺失时使用站点现有回退行为。
- 侧栏折叠只影响展示，不改变菜单权限和页面数据。
- 移动端底部导航固定尺寸，避免遮挡内容；内容区预留对应底部空间。

## 验收方式

根据项目约束不执行 `npm run build` 或自动化测试。实现后启动现有 Dever 环境，使用浏览器分别检查桌面和移动视口：

1. 三个菜单均跳转到真实 URL，刷新和浏览器前进后退正常。
2. 桌面外壳的结构、尺寸、字体和配色与 `bot/body` 一致。
3. 侧栏折叠后内容区域正确扩展。
4. 移动端显示底部导航，页面内容不被遮挡。
5. 线索录入、搜索、转客户以及客户列表详情等现有操作仍可使用。

## 明确不做

- 不新增 CRM 功能或菜单。
- 不调整后端接口、数据库或权限模型。
- 不改后台管理端样式。
- 不复制 `bot/body` 中与 CRM 无关的新建、积分、社区等入口。
